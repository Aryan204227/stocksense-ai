1. Introduction

What is the Stock Market?

  • A global platform where people buy and sell shares of public companies.
  • Prices fluctuate based on company performance, news, and world events.

Example: If you buy an Apple share for $100, and good news drives the price to $110, you make a profit.

2. Problem Statement

  • Information Overload: Thousands of articles are published daily. Humans cannot read them all.
  • Emotional Trading: Retail investors often panic and sell based on a single bad headline.
  • Fragmented Tools: Users must use multiple apps to check prices and read news separately.

3. Objective

  • Build a single unified platform for live stock prices and news.
  • Use Artificial Intelligence to automatically read news and calculate a Sentiment Score (Positive/Negative).
  • Help users make smart, safe, and data-driven trading decisions instantly.

4. Key Features

  • Global Search: Find any company stock instantly.
  • Real-Time Data: Live, millisecond accurate prices.
  • Live Charts: Interactive 30-day historical graphs.
  • AI Sentiment: Auto-reads news & scores market mood.

5. Technologies Used

  • Frontend: React.js, Tailwind CSS, Recharts (Graphs)
  • Backend: Node.js, Express.js, REST APIs
  • Hosting: Render Cloud Infrastructure

6. API Integration

  • Yahoo & Finnhub APIs: Used to fetch live stock prices and history charts securely.
  • NewsAPI: Fetches the top 10 latest news articles about the company.
  • Fallback System: If one API crashes, the system instantly switches to another so the app never goes down.

7. System Architecture

React UI → Node Backend → Live APIs

User Searches → Backend Fetches Data → AI Processes Text → UI Renders Charts

8. Working (Step-by-Step)

  1. User opens the web application.
  2. Types a company name in the search bar (e.g., "Google").
  3. System securely fetches live prices and recent news.
  4. AI Engine scans the news text and generates a Sentiment Score.
  5. Dashboard updates instantly with charts and final verdict.

9. Target Audience

  • Beginner Investors: Simplifies complex financial news into an easy-to-understand Positive/Negative score.
  • Finance Students: A practical tool to study how real-world news impacts stock market sentiment.
  • Casual Traders: Provides quick, on-the-go insights without needing to read dozens of articles daily.

10. Advantages

  • Saves Time: AI summarizes 50 articles instantly.
  • Easy to Use: Clean, intuitive design.
  • Better Decisions: Prevents emotional panic trading.
  • Always Available: Accessible from laptop or mobile.

11. Challenges Overcome

  • API Rate Limiting: Free financial APIs crashed when queried too often. We built a robust fallback mechanism to switch providers dynamically.
  • Production Deployment: Resolved complex CORS errors and backend routing issues to make the full-stack site live on Render.
  • Async Data Sync: Ensuring the AI sentiment analysis perfectly syncs with live price charts without lagging the user interface.

12. Limitations

  • Limited Data Availability: The system currently operates on limited present data and recent news, lacking decades of extensive historical data.
  • Future Unpredictability: While AI analyzes current sentiment well, it cannot perfectly predict future market behavior or sudden unprecedented events.
  • Scope of Analysis: Currently focuses on immediate news impact, lacking deeper fundamental financial analysis (like balance sheets).

13. Future Scope

  • User Accounts: Allow users to login and save custom Watchlists.
  • Price Alerts: Send Email/SMS when a stock crashes.
  • ChatGPT Integration: AI generates a detailed human summary of stock drops.
  • Data Expansion: Integrate premium data sources for deep historical trend analysis.
  • Mobile App: Convert the web platform into an Android and iOS application.

14. Conclusion

StockSense AI bridges the gap between complex financial data and human understanding.

By combining live stock prices with AI news sentiment, we provide a fast, easy, and data-driven tool for modern investors.
