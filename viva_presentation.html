The Problem

Information Overload

Thousands of financial articles are published daily. It's impossible for a human to read and analyze them all quickly.

Emotional Trading

Retail investors often make rushed decisions in panic based on a single bad headline, ignoring the big picture.

Fragmented Data

Users have to check 5 different websites to see stock charts, read news, and understand market trends.

The Solution

An AI-powered aggregator that automates market research.

  • Fetches live news from top financial sources automatically.
  • Analyzes the sentiment using AI (Positive, Negative, or Neutral).
  • Correlates the news sentiment directly with live stock market prices.
  • Visualizes everything on one single, stunning, easy-to-use dashboard.

Key Features

AI Sentiment Engine

Scans articles and calculates confidence scores (0-100%) to declare market mood.

Live Price Tracking

Real-time quotes and interactive 30-day historical trend charts.

Smart API Fallbacks

Automatic failover mechanism (Yahoo → Finnhub) ensures 99% uptime.

Live vs Demo Mode

Instantly toggle between real production data and safe simulation mode.

Technology Stack

Frontend

  • React.js & Vite: Ultra-fast UI rendering
  • Tailwind CSS: Custom Glassmorphism design
  • Recharts: Dynamic financial graphing
  • Framer Motion: Fluid component animations

Backend

  • Node.js & Express: Fast API routing
  • Axios: Secure external data fetching
  • NewsAPI & Finnhub: Live market data sources
  • Render: Cloud production deployment

How It Works

1

User Searches

User types "Tesla" on the React interface.

2

Backend Fetches Data

Node.js server contacts Yahoo Finance & NewsAPI simultaneously.

3

AI Processing

NLP algorithms scan the news text to generate a Sentiment Score.

4

Dashboard Render

Combined data is sent back to React, triggering chart animations.

Conclusion

StockSense AI successfully bridges the gap between raw financial data and human understanding, providing a powerful, fast, and automated tool for modern retail investors.
